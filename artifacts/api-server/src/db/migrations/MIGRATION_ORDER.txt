Reihenfolge für PostgreSQL (Produktion / frische DB)
====================================================
Alle Dateien sind idempotent (IF NOT EXISTS / ADD COLUMN IF NOT EXISTS), wo möglich.
Am zuverlässigsten: Deploy-Skript aus dem Repo-Root:

  DATABASE_URL="postgresql://…" bash scripts/deploy-onroda-production.sh --only-migrations

Manuell (gleiche Sortierung wie das Skript — nach Dateinamen 001_, 002_, …):

  cd /pfad/zu/imoove/artifacts/api-server
  export DATABASE_URL="postgresql://…"

  for f in src/db/migrations/[0-9][0-9][0-9]_*.sql; do
    psql "$DATABASE_URL" -v ON_ERROR_STOP=1 -f "$f"
  done

  # Tracker (optional, wenn ihr das Deploy-Skript nutzt — sonst fehlt onroda_deploy_migrations):
  # Das Skript deploy-onroda-production.sh trägt nach jedem -f automatisch in onroda_deploy_migrations ein.
